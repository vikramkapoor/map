Detailed Description

template<class NodeT, class LeafNodeT = NodeT>
class sparta::statistics::ReportStatisticsHierTree< NodeT, LeafNodeT >

Helper class used to build up a "report statistics hierarchy tree", where the template types are: 

NodeT     -- This is the class you want instantiated for each
             sparta::Report node (which includes subreports)
LeafNodeT -- This is the class you want instantiated for each
             sparta::StatisticInstance (at the leaves / no children)

For example: ReportStatisticHierTree<BigNode, LittleNode> tree(r); 

 // where the report 'r' looks like:

                 r
     -------------------------
     |           |           |
    top      scheduler      meta
  -------      |   |
   |   |      tix mss
  foo bar

In this use case, your hiearchy tree would be created as: 

 BigNode 'r'
    - children_:
           BigNode 'top'
              - children_:
                     LittleNode 'foo'
                     LittleNode 'bar'
           BigNode 'scheduler'
              - children_:
                     LittleNode 'tix'
                     LittleNode 'mss'
           LittleNode 'meta'
