Detailed Description

template<class PointerT>
class sparta::SpartaSharedPointer< PointerT >

Used for garbage collection, will delete the object it points to when all objects are finished using it.

Simple, thread-unsafe class that will delete memory it points to when the last memory reference is deconstructed.

Example usage:

void foo()
{
sparta::SpartaSharedPointer<int> ptr(new int);
int * t = new int(0); // Don't do this, BTW
sparta::SpartaSharedPointer<int> ptr2(t);
ptr2 = ptr;
*ptr = 5;
sparta_assert(*ptr2 == 5);
sparta::SpartaSharedPointer<int> ptr3(new int(10));
sparta_assert(*ptr3 == 10);
// Don't delete!
}

This class can be used independently, or more efficiently with sparta::allocate_sparta_shared_pointer<T>. See sparta::SpartaSharedPointer::SpartaSharedPointerAllocator for more information.
